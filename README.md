# Backend & API Projects

This repository contains a collection of backend and API-focused projects built with
JavaScript using Node.js and Express. The projects demonstrate practical experience in
building REST APIs, authentication systems, server-side rendered applications, and
integrations with databases and third-party services.

---

## Tech Stack

- JavaScript (ES6+, Node.js runtime)
- Express
- REST APIs
- Authentication & Authorization (JWT, OAuth, Sessions)
- PostgreSQL (pg)
- MongoDB & Mongoose
- Server-side rendering (EJS)
- Stripe API
- Environment variables (dotenv)
